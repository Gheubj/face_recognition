Нейросеть, распознающая человека по биометрии 

Для использования нейросети нужно пройти 3 шага, для каждого есть отдельная программа:

1. face_gen  - Пользователь вводит свой id, фотографии с его камеры добавляются в датасет. 

2. face_train - Нейросеть обучается по фото из датасета и запоминает пользователя.

3. face_detect - Основная программа распознает человека в реальном времени и показывает его id, нейросеть подтверждает личность человека при сходтсве на определенное количество процентов.



P.S.
    Данная нейросеть не доработана до идеала, может быть не точной, не гарантируется 100% распознование пользователя, т.к. является проектом
